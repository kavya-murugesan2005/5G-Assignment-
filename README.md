# 5G Fundamentals and Architectures – Open5GS Assignments

This repository contains two practical assignments completed as part of the **5G Fundamentals and Architectures (EC23712)** course. The assignments focus on deploying and analyzing software-based LTE/5G networks using **Open5GS**, **srsRAN**, and various networking tools.

---

## 📂 Assignments Included

### 📌 Assignment 1: Sustained 5-Minute TCP Transfer & Throughput Stability Analysis

**Description**
- Configured the Open5GS 5G Core Network.
- Generated a continuous 5-minute TCP transfer using **iperf3**.
- Collected throughput data and analyzed network performance.
- Calculated average throughput, peak throughput, minimum throughput, and jitter.
- Visualized throughput stability using Python and Matplotlib.

**Tools & Technologies**
- Ubuntu 22.04
- Open5GS
- MongoDB
- iperf3
- Python
- NumPy
- Matplotlib

**Key Outcomes**
- Evaluated long-duration TCP performance.
- Measured throughput stability and jitter.
- Generated performance graphs for analysis.

---

### 📌 Assignment 2: LTE Attach Procedure Verification using srsRAN & Open5GS

**Description**
- Built a live LTE network using **srsRAN** and **Open5GS EPC**.
- Verified the LTE Attach Procedure.
- Analyzed S1AP, NAS, SCTP, GTP-U, and Diameter protocols.
- Validated subscriber authentication.
- Verified end-to-end connectivity using a Ping test.

**Tools & Technologies**
- Ubuntu Linux
- Open5GS EPC
- srsRAN
- MongoDB
- Open5GS WebUI
- Wireshark

**Key Outcomes**
- Successful LTE Attach Procedure.
- Verified LTE interface/protocol mapping.
- Confirmed subscriber authentication.
- Validated user-plane connectivity.
